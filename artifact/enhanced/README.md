
# Enhanced Artifact

This folder contains the enhanced version of my CS 340 Project Two dashboard, modified during CS 499 to demonstrate improvements in software design, algorithm efficiency, database architecture, and security practices.

## Running the Enhanced Version

1. Install dependencies:
   pip install -r requirements.txt

2. Set environment variable:
   export MONGO_URI="your_connection_string"

3. Run:
   python ProjectTwoDashboard_enhanced.py
